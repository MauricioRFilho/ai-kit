# ARCH_DECISIONS.md

Registro de decisões arquiteturais do projeto.

Use uma entrada por decisão.

---

## ADR-001 — Exemplo

### Status
Aceita

### Contexto
Precisávamos lançar um MVP rápido com baixo custo operacional.

### Decisão
Usar monólito modular em vez de microsserviços.

### Consequências
- deploy mais simples
- menor custo operacional
- menor complexidade inicial
- escalabilidade organizacional menor no longo prazo, aceitável para o estágio atual
